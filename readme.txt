# Spy Cat Agency Management System

This is a FastAPI application designed to manage the operations of the Spy Cat Agency (SCA). The system allows users to manage spy cats, missions, and targets efficiently.

## Features

- Create, read, update, and delete spy cats.
- Create, read missions.
- Manage targets associated with each mission.
- Mark missions and targets as complete.

## Technologies Used

- Python 3.8+
- FastAPI
- SQLAlchemy
- SQLite
- Uvicorn (ASGI server)
